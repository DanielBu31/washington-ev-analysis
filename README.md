##Washington-ev-analysis

-Washington State Electric Vehicle Population Analysis | Python Data Analysis | Portfolio Project
-I analyzed Washington State's electric vehicle data to see how EV adoption is growing and where the most EVs are located.

## What I Did
- Cleaned the raw data from data.wa.gov
- Looked at how EV registrations have changed over time
- Found the most popular EV brands
- Mapped where EVs are concentrated across the state
- Compared battery electric vs plug-in hybrid vehicles

## What I Found
- EV adoption is growing really fast
- Tesla is the most popular brand by far
- King County (Seattle area) has the most EVs
- Most people choose battery electric over plug-in hybrid

- Python
- Pandas for data cleaning
- Matplotlib for charts
- Jupyter Notebook
